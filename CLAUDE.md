# CLAUDE.md

本文件为 Claude Code (claude.ai/code) 提供项目开发指引。

## 项目概述

这是一个纯Python实现的TOTP临时密码生成器，与微信小程序版本完全兼容，可生成3-60分钟有效期的一次性密码。

## 技术栈
- Python 3.7+
- 无外部依赖，加密算法使用Python标准库实现

## 开发规范
- 保持算法逻辑与微信小程序版本100%兼容，生成的密码必须完全一致
- 所有修改必须通过现有测试用例
- 新增功能时必须补充对应的测试用例

## 核心文件说明
| 文件 | 功能 |
|------|------|
| `generate_totp.py` | 核心算法实现，使用Python标准库的SHA1、HMAC-SHA1实现TOTP生成逻辑 |
| `test_totp.py` | 测试用例集合，包含固定时间点的密码验证 |

## 常用命令
```bash
# 运行测试用例
uv run pytest

# 生成密码
python3 generate_totp.py <管理员密码> <有效期>
```

## 注意事项
- 重构代码时必须保持算法逻辑完全不变，测试用例必须全部通过
- 不要添加任何外部依赖，保持项目零依赖特性
- 密码生成规则固定为10位，前8位为密码内容，后2位为有效期，请勿修改
- API更新：`generate_totp()` 和 `generate_digit_password()` 现在接受整数类型的seed参数，不再需要传入数字数组
- CLI保持完全向后兼容，命令行输入仍然使用字符串格式的管理员密码
- 日期计算逻辑包含历史兼容性实现，禁止修改，必须保持与原手动累加逻辑100%一致
- 密码生成采用低位在前的顺序，与常规TOTP实现不同，需严格保留原有生成逻辑
- 所有修改必须保证与微信小程序版本生成的密码完全一致，兼容性优先于性能或代码优美
